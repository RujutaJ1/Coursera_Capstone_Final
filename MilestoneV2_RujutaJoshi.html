Project Objective and Milestone Objective

The main project is to create an application which can predict the most probable next words basis the user inputs. The prediction algorithm will use the data provided by SwiftKey to learn about the probable phrases, and predict the next words.

The objective of the milestone report, is to perform the primary steps of downloading, and staging the data, perform the exploratory analysis and describe the approach towards the main project.

Exploratory Data Analysis Methodology

The Exploratory Data Analysis (EDA) methodology for this project will be to:

  1. Load the text data into corpora.

  2. Clean the corpora by removing extra white space, putting text all in lower-case, removing numbers, removing punctuation. In the final product, profanity will be removed.

  3. Understand the size ,length of the data, frequency of the words, and most often used words.

  4. Create n-gram tokens.

  5. develop histograms to understand more commonly used words.

The Approach for the main Project

The eventual aim is to derive a compact table of frequency count for all terms (single words and word pairs) small enough to be loaded into the Shiny server.

The system will use Markov prediction based on the probabilities of bigrams and trigrams in the combined data sets. Markov prediction determines what the next likely word based on shorter n-grams rather than whole sentences. This will perform better than attempting to assign probabilities to whole phrases, as the data sets cannot contain all possible sequences of words that could be entered by the user.

Looking at the N-gram for the sample.

We first tokenise the sample into unigrams, bigrams and trigrams this will help us understand the distribution of the words

UnigramTokenizer <- function(x) NGramTokenizer(x, Weka_control(min = 1, max = 1))
en_us.sample.unigram <- TermDocumentMatrix(totalsample, control = list(tokenize = UnigramTokenizer))
FreqTerms <- findFreqTerms(en_us.sample.unigram, lowfreq = 20)
en_us.sample.frequency.vector.uni <- sort(rowSums(as.matrix(en_us.sample.unigram[FreqTerms,])),decreasing=TRUE)
en_us.sample.frequency.dataframe.uni <- data.frame(word = names(en_us.sample.frequency.vector.uni),freq=en_us.sample.frequency.vector.uni)

qplot(data=en_us.sample.frequency.dataframe.uni[1:50,], x=word, y=freq, geom="bar", stat="identity") + theme(axis.text.x=element_text(angle=90)) + labs(title="Top 50 Unigrams with a frequency greater than 20") + labs(x="Unigram") + labs(y="Frequency")
